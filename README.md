## Features

- **Responsive layout** — desktop grid hero, collapsible navigation on small screens  
- **Phosphor icons** — consistent icon set via CDN  
- **Scroll reveal** — sections fade in (respects `prefers-reduced-motion`)  
- **FAQ accordion** — one panel open at a time  
- **Optional social URLs** — edit `site-config.js`; leave values empty for **placeholder-only** buttons (no navigation), or add your links for a live site  
- **Back-to-top** control  

---

## Tech stack

HTML5 · CSS3 (custom properties, grid, flex) · JavaScript (ES5-style IIFE for broad compatibility) · [Phosphor Icons](https://phosphoricons.com/) · Google Fonts (DM Sans, Fraunces)

## Getting started

1. **Clone** this repository (or download ZIP).  
2. Open **`index.html`** in a browser, or serve the folder with any static server.  
3. **Customize** copy, images under `images/`, and colors in `style.css`.  
4. **Social links (optional):** open **`site-config.js`** and set URLs for keys like `github`, `youtube`, `linkedin`, etc. Empty strings keep buttons visual-only (good for a template you share).  

### GitHub Pages

Push to a GitHub repo → **Settings → Pages →** deploy from branch (usually `main` / root or `/docs`). Your live URL will look like:  
`https://<username>.github.io/<repo>/`
